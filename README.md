# Como executar o projeto
 
Pré-requisitos: JavaSE 17

## Passo 1

```bash
# clonar repositório
git clone https://github.com/Alendrew/workshop-springboot3-jpa.git

# entrar na pasta do projeto back end
cd workshop-springboot3-jpa

# executar o projeto
./mvnw spring-boot:run
```
